QingEditor 是一套开源轻量级的在线HTML编辑器，主要用于让用户在网站上获得所见即所得编辑效果，开发人员可以用 QingEditor 把传统的多行文本输入框(textarea)替换为可视化的富文本输入框。 QingEditor 使用 JavaScript 编写，可以无缝地与 Java、.NET、PHP、ASP 等程序集成，比较适合在 CMS、商城、论坛、博客、Wiki、电子邮件等互联网应用上使用。 

 本人也是网站爱好者，经常在网上找好用的在线编辑器。 但是找到的，不是不兼容，就是太大，运行太卡。 所以综合了一下，自己开了一个最简单的在线编辑器，对于一些简单的文字编辑是足够使用了。 最主要是轻，相对百度编辑器那种动不动几十兆的程序来说。 最后希望大家能喜欢哦 




一 、下载小清编辑器Qing Editor。

1、官方网站下载 : http://www.mlmzj.com/mlmzj/demo/qingeditor/qingeditor.rar 

 2、站长之家下载 : http://www.mlmzj.com/mlmzj/demo/qingeditor/qingeditor.rar 
二 、在网页文件里添加代码。

1、在需要添加编辑器的控件，比如textarea控件，加上id="Qing_input"(注意区分大小写)。 


<textarea id="Qing_input" name="content"> 

 2、在编辑器控件的下面添加以下代码，路径自己修改。 


<link rel="stylesheet" href="qingeditor/qing-style.css" /> 
 <script type="text/javascript" src="qingeditor/qing-editor-all.js"></script> 

 3、最一步，给要提交的表单加上editor.post();否则将接收不到编辑器的数据，切记。 


<form action="?mydo=addok" name="form1" method="post" onSubmit="editor.post();"> 

 4、一个标准的小清编辑器基础文档，qing-style.css和qing-editor-all.js的路径需自行设置好 



<!doctype html>
 <html>
 <head>
 <meta charset="gb2312">
 <title>小清编辑器</title>
 </head>
 <body>


 <form action="?mydo=addok" name="form1" method="post" onSubmit="editor.post();">
 <textarea id="Qing_input" name="content"></textarea>
 </form>


 <link rel="stylesheet" href="qingeditor/qing-style.css" /> 
 <script type="text/javascript" src="qingeditor/qing-editor-all.js"></script>


 </body>
 </html>
